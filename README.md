# EFA Label PDF Generator

وب‌اپ تولید PDF لیبل انبار از روی فایل Excel (transfers.xlsx) با خروجی دو لیبل در هر صفحه (2-up)،
مشابه فایل نمونه `barcode.pdf`.

## تکنولوژی‌ها

- React + Vite + TypeScript
- Tailwind CSS
- SheetJS (xlsx) برای خواندن Excel
- pdf-lib برای ساخت PDF
- JsBarcode برای ساخت بارکد میله‌ای (CODE128)
- Canvas API برای رندر لیبل به‌صورت تصویر و جاسازی آن در PDF

## ورودی

یک فایل Excel با حداقل ستون‌های زیر (هدرها می‌توانند کمی متفاوت باشند):

- `نام کالا` (یا معادل آن)
- `کد انبار کالا` (یا معادل آن)
- `تعداد`

ستون‌ها براساس نام فارسی و معادل انگلیسی به‌صورت خودکار تشخیص داده می‌شوند.

برای هر ردیف، به تعداد مقدار ستون «تعداد»، لیبل ساخته می‌شود.

## خروجی

- یک فایل PDF با اندازه صفحه 565 × 141 پوینت
- در هر صفحه دو لیبل کنار هم (2-up) برای رول لیبل‌پرینتر دو-ستونه
- هر لیبل شامل:
  - بارکد میله‌ای CODE128 از روی «کد انبار کالا»
  - متن کد زیر بارکد
  - نام کالا زیر کد (به‌صورت راست‌به‌چپ روی Canvas)

## نحوه اجرا

```bash
npm install
npm run dev
```

سپس در مرورگر به آدرس (پیش‌فرض):

- http://localhost:5173

بروید.

## نکات

- تمام پردازش (خواندن Excel، ساخت بارکد، تولید PDF) روی مرورگر و به‌صورت آفلاین انجام می‌شود.
- ابعاد صفحه PDF از روی فایل نمونه `barcode.pdf` استخراج شده تا با رول فعلی شما سازگار باشد.
